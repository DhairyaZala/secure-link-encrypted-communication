Secure-Link: Encrypted Communication Suite

A cybersecurity project implementing a hybrid encryption-based secure communication system.

 Overview

Secure-Link is a secure communication prototype that combines symmetric and asymmetric cryptography to ensure confidentiality, integrity, and authentication in data transmission.

This project demonstrates practical implementation of modern cryptographic techniques used in real-world secure systems.

---


Features

- Hybrid Encryption (RSA + DES)
- Diffie-Hellman Key Exchange
- SHA-256 Hashing for Integrity Verification
- Digital Signature for Authentication
- Simulated Secure Client-Server Communication

 How It Works

1. Diffie-Hellman is used to establish a shared session key.
2. RSA encrypts the symmetric DES key securely.
3. DES encrypts the message for fast data transmission.
4. SHA-256 ensures message integrity.
5. Digital signatures verify sender authenticity.
